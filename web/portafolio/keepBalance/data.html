Keep Balance

10/10/2015

Proyecto Universitario para la Detección Automáticas de Caídas, Monitoreo y Alertas Médicas en Tiempo Real.

El proyecto está formado por una aplicación móvil que sirve para censar y envía los datos obtenidos a un servidor.

Además se desarrolló una aplicación web para la presentación de los datos y generación de alertas y reportes.
